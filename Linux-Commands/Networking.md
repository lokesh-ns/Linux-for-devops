Networking

1.	How do you find the IP address of your system?
● ip addr or ifconfig

2.	How do you check active network connections?
● ss -tuln or netstat -tuln

3.	How do you ping a host to check connectivity?
● ping host

4.	How do you test port connectivity?
● telnet hostname port

5.	How do you display routing information?
● route

6.	How do you check open ports on your system?
● lsof -i

7.	How do you download files from the internet?
● wget URL

8.	How do you transfer files between systems using SCP?
● scp source_file user@remote_host:/destination_path

9.	How do you use SSH to connect to a remote server?
● ssh user@remote_host

10.	How do you set up a static IP address?
● Edit the network configuration file (e.g., /etc/network/interfaces)
