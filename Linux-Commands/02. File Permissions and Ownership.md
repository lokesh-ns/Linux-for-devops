# File Permissions and Ownership

1.	How do you change file permissions?
-   chmod permissions filename

2.	How do you change file ownership?
-   chown user:group filename

3.	How do you view file permissions in a directory?
-   ls -l or ls -ltr

4.	How do you recursively change permissions for a directory?
-   chmod -R permissions directory_name

5.	What does the chmod 777 command do?
-   Grants read, write, and execute permissions to everyone

6.	How do you set permissions for a script to be executable?
-   chmod +x script.sh
