Process Management

1.	How do you view running processes?
-   ps -aux

2.	How do you kill a process by PID?
-   kill PID

3.	How do you find the PID of a running process by name?
-   pgrep process_name

4.	How do you display processes in real-time?
-   top or htop

5.	How do you stop a process gracefully?
-   kill -15 PID

6.	How do you stop a process forcefully?
-   kill -9 PID

7.	How do you start a background process?
-   command &

8.	How do you monitor resource usage by a process?
-   pidstat

9.	How do you view system logs for a process?
-   journalctl -u service_name

10.	How do you create a custom alias for a command?
-   alias alias_name='command'

11.	How do you remove an alias?
-   unalias alias_name

12.	How do you run a command as another user?
-   sudo -u username command

13.	How do you list all environment variables?
-   printenv or env

14.	How do you export a variable in Linux?
-   export VAR=value
