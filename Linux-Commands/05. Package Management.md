# Package Management

1.	How do you install a package on Debian-based systems?
-   apt install package_name

2.	How do you update all packages on a Debian-based system?
-   apt update && apt upgrade

3.	How do you remove a package on Debian-based systems?
-   apt remove package_name

4.	How do you install a package on RHEL-based systems?
-   yum install package_name or dnf install package_name

5.	How do you remove a package on RHEL-based systems?
-   yum remove package_name or dnf remove package_name

6.	How do you search for a package in Debian repositories?
-   apt search package_name

7.	How do you list all installed packages?
-   dpkg -l (Debian) or rpm -qa (RHEL)

8.	How do you check which package a file belongs to?
-   dpkg -S filename (Debian) or rpm -qf filename (RHEL)

9.	How do you clean up cached packages on Debian-based systems?
-   apt clean

10.	How do you enable or disable a repository in RHEL-based systems?
-   yum-config-manager --enable repo_name or --disable repo_name
